# Localens

Premium product and service discovery platform where users can search for any product, business, or service and get rich results including an interactive map, images, ratings, user feedback, and an AI-generated summary.

## Tech Stack

- **Frontend**: Next.js 14+, TypeScript, Tailwind CSS v4, shadcn/ui
- **Backend**: Next.js API Routes, MongoDB Atlas, Mongoose
- **Auth**: Clerk
- **Cache**: Upstash Redis
- **AI**: Groq API (LLaMA 3 70B)
- **APIs**: Foursquare Places, Unsplash, Mapbox
- **Deployment**: Vercel

## Features

- Smart search with autocomplete
- Interactive dark map (Mapbox)
- AI-generated summaries and chat
- User reviews with sentiment analysis
- Bookmark saved places
- Fully responsive design
